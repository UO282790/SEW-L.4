Diferencias y similitudes entre Typescript y Javascript

Al estar basado en Javascript, Typescript hereda gran cantidad de sus características. Sin embargo, existen algunas diferencias que merece la pena destacar:

Comparación entre Typescript y Javascript
Typescript Javascript
Lenguaje orientado a objetos Leguaje de scripting en el cliente
Soporta tipado estático No soporta tipado estático
Busca errores en tiempo de desarrollo Solo busca errores en tiempo de ejecución
Se traduce a Javascript en tiempo de compilación Se ejecuta directamente en el navegador
Soporta genéricos e interfaces No soporta ni genéricos ni interfaces
