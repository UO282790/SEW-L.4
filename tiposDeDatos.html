Tipos de datos de Typescritp

Typescript, a diferencia de Javascript, incluye los siguientes tipos de datos que se comprueban de forma estática:

Número
Se usa para representar tanto números enteros como números decimales.
Boolean
Se usa para representar valores booleanos (verdadero y falso).
String
Se usa para representar cadenas de texto.
Void
Se usa para el tipo de retorno de algunas funciones.
Null
Se usa cuando un atributo no tiene un valor asignado.
Undefined
Se usa para indicar que un atributo no ha sido inicializado.
Any
Se usa cuando un atributo va a contener (o pueda contener) cualquier tipo de valor.
